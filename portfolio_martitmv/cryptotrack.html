About

Cryptotrack API is a web scraper that collects cryptocurrency news articles and live prices from various sources, stores and displays them.

It uses the axios library to send HTTP requests to the specified sources, the cheerio library to parse the HTML responses, and the express library to set up a server.

The API has two sections, "/Price" & "/News".

In "/PRICE" section it scrapes the latest coin prices and 24-hour volumes, so users can track the performance of their favorite cryptocurrencies.

In "/NEWS" section it provides a convenient way for users to stay informed about the latest news in the crypto space and also links as source for the articles it reads.

All the information provided by the API is displayed in JSON format, ensuring easy integration with any application or platform.
